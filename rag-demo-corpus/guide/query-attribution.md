# Query Attribution Guide

## Regional Growth Workbook

The workbook titled Regional Growth explains rollout posture for the staged territory expansion plan. This section is written for the exact titled-workbook phrasing so the base query can land on a stable source before any heuristic rewrite runs.

Operators also describe the same evidence with more generic spreadsheet language. They ask for the worksheet named Regional Growth, the workbook, the worksheet, the spreadsheet, and the xlsx source when they are tracing why rollout posture changed across the territory plan.

The Regional Growth worksheet keeps the market expansion view, approval notes, and rollout posture in one place. The spreadsheet wording stays intentionally repetitive here so the variant query can reinforce the same section instead of drifting into a different source family.

Auditors reviewing attribution should see this section survive both the preserved titled-workbook query and the expanded spreadsheet variants. That makes the section a good benchmark for mixed query attribution rather than a pure primary-only or variant-only result.
