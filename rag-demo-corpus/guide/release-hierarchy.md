# Release Operations

This guide introduces the release control workflow for nested section navigation.

## Stable Lane

The stable lane holds the promoted retrieval baseline and tracks handoff readiness.

### Approval Gates

Approval gates should remain visible when a candidate needs manual promotion review.

### Incident Review

Incident review should preserve section-level evidence and recent release actions.

## Canary Lane

The canary lane records early rollout posture and drift before stable promotion.

### Handoff Drift

Handoff drift should stay linked to the lane-specific section that raised the alert.
